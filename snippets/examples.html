E-mail address link

The following should be converted in a valid email link. "email" class is preserved for e.g. microformats:

<span class="email">myname [at] my domain</span> → myname [at] my domain

The following should ALSO be converted, JS uses extra options

<span class="email2">JohnDoe *AT* johns domain</span> → JohnDoe *AT* johns domain
